# Java Immersion Course by Accenture
## Sprint 1
A test API implementation done as an assessment for the course.
It contains only four endpoints and is documented using the openapi standard (swagger)

### Base URL
http://localhost:8080/api

### Endpoints
1. GET /greeting
2. POST /submit
3. GET /user/{id}
4. GET /search?query=value

### Open API Documentation
http://localhost:8080/swagger-ui/index.html

http://localhost:8080/api-docs
